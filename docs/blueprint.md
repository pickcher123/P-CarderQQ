# **App Name**: P+Carder 數字收藏

## Core Features:

- 首頁: 具有 3D 浮動卡片動態背景、標語「收藏未來」以及快速訪問熱門卡池的入口。
- 抽卡專區: 主題卡池、卡包的箔包視覺效果、即時掉落率和剩餘庫存顯示。
- 卡片開包動畫: 逼真的開包模擬，包括震動、爆炸和顯示動畫。
- 拼卡專區: 用於卡片投注的財富輪盤界面，帶有可調節的獲勝率滑塊。
- 滿團福袋（完整團體幸運袋）: 幸運袋購買平台，顯示進度條以及當前和總參與者、買家的 1-100 網格，並在完成時觸發「揭示彩票」。
- 收藏庫: 一個個人卡片庫存，您可以在其中管理您的數字卡片收藏，具有稀有度過濾器和批量運輸選項。快速出售時提供即時價值。
- 會員中心: 會員個人資料，顯示用戶等級、購買金額和其他統計信息。付款集成（使用模擬網關）和地址簿集成。

## Style Guidelines:

- 主色：霓虹藍 (#06B6D4)，用於主要按鈕和互動。喚起未來科技和活動的感覺。
- 強調色：傳奇金 (#EAB308)，用於高價值操作和突出顯示。類似於霓虹藍，但具有對比鮮明的亮度和飽和度。
- 危險色：迷人粉 (#DB2777)，用於投注功能和警告。作為一種具有高可見性的補充。
- 底色：深午夜藍 (#0F172A)，帶有碳纖維紋理作為背景。這種色調將產生最大的對比度，在色輪上非常接近霓虹藍和迷人粉。
- 標題/Logo 字體：'Orbitron'（無襯線字體），具有科技感。注意：目前僅支持 Google Fonts。
- 數據/序列號字體：'Rajdhani'（無襯線字體），具有工業和精確的感覺。注意：目前僅支持 Google Fonts。
- 正文/中文字體：'Noto Sans TC'（無襯線字體），具有現代感。注意：目前僅支持 Google Fonts。
- 使用適合主題的發光霓虹風格圖標，這些圖標根據稀有度而變化。
- 3D 卡片基於懸停的光澤效果，具有各種顏色的發光，以及翻開時的 Y 軸翻轉。
- 科幻微調器，在數據加載時顯示，標題為「正在讀取區塊鏈數據...」。
- 優化桌面（每行 5 張卡片）、平板電腦（每行 3 張卡片）和手機（每行 2 張卡片）的網格視圖。
