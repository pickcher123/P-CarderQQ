# P+Carder 遊戲區域視覺風格計畫書 (Game Style Plan v7.4)

這份計畫書定義了 P+Carder 四大遊戲專區的視覺 DNA，將「賽博未來」與「像素復古」完美融合，營造獨特的遊戲化收藏體驗。

---

## 1. 抽卡專區 (Draw Zone)
**核心主題：P+ Handheld Console (掌上遊戲機)**
- **機身建模**：採用深靛藍 (`indigo-900`) 與深紫拼接，模擬 8-bit 時代掌機的厚重塑料質感。
- **邊框細節**：使用 12px 像素階梯式立體邊框，四角點綴「像素螺絲孔」。
- **螢幕視覺**：
    - **LCD 模擬**：深色背景搭配微弱的翡翠綠電子光暈。
    - **掃描線**：動態 CRT 掃描線 (Scanlines)，營造復古螢幕深度。
- **操作元件**：模擬 A/B 彩色實體鍵（櫻桃紅與檸檬黃），左側 D-Pad 十字鍵裝飾。
- **進度回饋**：像素化「血條」設計，隨庫存減少由「青藍」轉「琥珀」再轉「危險紅」。

## 2. 拼卡專區 (Bet Zone)
**核心主題：Casino Slot Machine (豪華拉霸機)**
- **機身建模**：深酒紅 (`#2d060e`) 搭配磨砂金屬黑，營造高端、沉穩的博弈氛圍。
- **邊框細節**：琥珀金像素裝飾線條，強調競技的奢華感。
- **螢幕視覺**：
    - **開獎轉輪**：具備深度遮罩陰影與兩側紅色動態指針，模擬機械轉動張力。
    - **售罄狀態**：像素化「斷電」灰階特效，斜跨醒目的粉色像素標籤。
- **操作元件**：
    - **啟動桿**：巨大的沉穩緋紅 (`#8b1a2a`) 實體啟動鍵，帶有環境光擴散特效。
    - **號碼網格**：1-10 數字採用具備下壓視覺回饋的實體像素按鍵。

## 3. 福袋專區 (Lucky Bag Zone)
**核心主題：Classic Game Boy Style (經典溫潤掌機)**
- **機身建模**：經典的溫潤灰色塑料質感 (`#d1d1d1`)，還原初世代掌機手感。
- **細節裝飾**：右下角 45 度角傾斜喇叭排孔，下方灰色長橢圓形 Select/Start 橡膠按鍵。
- **螢幕視覺**：
    - **冷光 LCD**：採用高對比冷灰色 (`#f0f4f7`) 介面，移除碗豆綠濾鏡以確保「卡片真實色彩」還原。
    - **點陣網格**：極輕量的像素點陣網格，維持數位螢幕的精密感。
- **操作元件**：深紫紅色 A/B 圓鈕，1-100 選號網格具備實體圓形按鍵的圓潤感與發光特效。

## 4. 團拆專區 (Group Break Zone)
**核心主題：Retro CRT TV & Home Console (復古電視與家庭遊樂器)**
- **機身建模**：復古胡桃木紋與石板灰拼接的 CRT 電視機，帶有轉盤式旋鈕與柵格狀喇叭。
- **螢幕視覺**：
    - **螢幕弧度**：影像位於帶有圓角與掃描線的螢幕內，模擬 CRT 的顯像管物理特性。
    - **訊號標示**：動態「LIVE」呼吸燈與訊號強度指針，強調「直播現場」感。
- **操作元件**：
    - **主機面板**：操作區重構成「Console Control Deck (遊樂器主機)」。
    - **實體按鈕**：採用模擬實體主機的排插式按鍵，購買鍵升級為「POWER ON」發光啟動開關。
- **結果呈現**：開獎名單採用「官方轉播頻道」風格，呈現專業的賽事結算質感。

## 5. 全站通用視覺規範
- **字體選用**：數據與數字統一使用 `Rajdhani` (工業窄體)；標題使用 `Orbitron` (科技感無襯線)；內文使用 `Noto Sans TC`。
- **交互回饋**：所有「實體按鍵」在點擊時必須具備 `translate-y-1` 的位移感與陰影變化。
- **層次控制**：即便在複雜的像素外殼下，核心卡片內容必須保持 `bg-card/40` 以上的透明度，確保在沉浸式背景下的閱讀性。
- **動態光影**：廣泛應用「全息掃光」與「電子脈衝」特效，為靜態的點陣風格注入生命力。
